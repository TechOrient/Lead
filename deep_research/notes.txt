Tüm adımlar
1. Kullanıcıdan gerekli olan promptun girilmesi
2. Kullanıcıya sistem tarafından brief verilmesi 
3. Bu brief doğrultusunda gerekli araştırmanın başlatılması 
4. Araştırılacak olan bilgiler.
    1. Şirket tam ad 
    2. Tam adres 
    3. Gerekli telefon numaraları 
    4. Çalışan kişi ve bilgiler /özellikle linkedin bilgisi
    5. Şirketin bulunduğu sektör
    6. Kuruluş yılı + kurucuların bilgisi
    7. Fabrikası ya da farklı bir üretim yerine sahip olduğunun bilgisi veya tesis bilgisi.
    8. Son olarak detaylı olacak olan rapor.
    9. Ekstra olarak sabit olan sitelerden belli başlı bilgileri getirmesi için araştırma yapacak

TÜM AGENT VE GEREKLİ AGENT MİMARİLERİ  :

1. Kullanıcı : Gerekli promptu giren ve de sonucu görecek olan kişi 
2. Supervisor Agent : Tüm agent'ların işlemlerini ve süreçlerini kontrol etme görevi
3. Prompt Parser Agent : Girilen promptu işleyen ve eksik bilgi var mı + brief olarak döndüren Agent
4. Web Search Agent : İnternet sitelerine bakan ve google search yapacak siteleri bulacak. 
5. Extract Agent : Sitelerden detaylı bilgileri bulacak olan agent 
6. Valide/Clean Up Agent : Tüm sonuçları inceleyen sonunda ise şirketimize uygun mu değil mi bunun kararını veren agent 
7. Rapor Agent: Bilgileri döküman haline getiren agent (son agent)
